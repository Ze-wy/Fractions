📄 ReadMe: Fraction Calculator (Python 3.11)

📌 Project Overview
A Python class for working with fractions. The project implements a Fraction class to store and manipulate rational numbers.

📝 Task
Create a "Fraction" class.

Storage: The class must store the numerator and denominator as its fields.
Methods: Implement methods for data input, data output, and access to individual fields (getters/setters).
Arithmetic: Create methods for performing arithmetic operations (addition, subtraction, multiplication, division, etc.).

✨ Features
Data Storage: Holds numerator and denominator.
Arithmetic Operations: Supports +, -, *, /.
Simplification: Automatically reduces results.
Encapsulation: Uses getters/setters for data access.

🐍 Tech Stack
Language: Python 3.11
Style: Object-Oriented Programming (OOP)
